# olena-salnikova-mars-open-api
Open API Project for Intro to Programming course with Code the Dream


# INSTRUCTIONS for running the project locally on your computer
(No other programs are needed. Just follow these steps and everything should work.)

1. Press the green "Code" button on the GitHub page.
2. Choose "Download ZIP" and save the file to your computer.
3. Find the ZIP file on your computer and open it.
4. Move the files from inside the ZIP to a new folder.
5. Find the file called `index.html` in that folder.
6. Double-click the `index.html` file.
7. The project will open in your browser and you can use it.
